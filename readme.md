# Database Tutorial 


#### Database Commands

##### Show Database
To see the databases are present with help of this commands we can see. 

```sql 
  show databases;
```

##### Create Database

```sql
    create database database_name;
```

##### Drop Database

```sql
    drop database databaseName; 
```

##### Change Database 

```sql
    use databaseName; 
```
